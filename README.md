# sensoren-gewaechshaus
Dieses Projekt realisiert ein IT‑gesteuertes Gewächshaus‑System für 100 C-Pflanzen mit zwei Raspberry Pi: Datenerfassung, Aktorsteuerung, lokale Speicherung, Alarmbenachrichtigungen, automatisierte Steuerlogik und sicherer Fernzugriff. Enthalten: kommentierte Python‑Skripte, Simulationsmodul, Installations‑/systemd‑Anleitungen, VNC‑Setup, Testplan.
